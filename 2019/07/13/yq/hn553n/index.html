vagrant创建管理虚拟机

安装前准备

  • Install the latest version of Vagrant.
  • Install VirtualBox

vargrant 使用

在任意目录下创建 vagrantfile 文件

在文件所在的目录下，执行vagrant up 创建并启动虚拟机， 执行vagrant ssh 连接进入到虚拟机控制台。
